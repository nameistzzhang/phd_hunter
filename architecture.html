ж ёеҝғз»„д»¶пғҒ

  1. CLI е…ҘеҸЈ (main.py)

    е‘Ҫд»ӨиЎҢдё»зЁӢеәҸпјҢжҸҗдҫӣдёүдёӘеӯҗе‘Ҫд»Өпјҡ

    • crawl: д»Һ CSRankings зҲ¬еҸ–ж•ҷжҺҲдҝЎжҒҜ

    • fetch-papers: д»Һ arXiv иҺ·еҸ–ж•ҷжҺҲи®әж–Ү

    • stats / list: жҹҘзңӢж•°жҚ®еә“еҶ…е®№

  2. зҲ¬иҷ«жЁЎеқ— (crawlers/)

    • CSRankingsCrawler: дҪҝз”Ё Selenium зҲ¬еҸ– CSRankings.org зҡ„еӨ§еӯҰжҺ’еҗҚе’Ңж•ҷжҺҲеҲ—иЎЁ

    • ArxivCrawler: дҪҝз”Ё arXiv API жҢүдҪңиҖ…жҗңзҙўи®әж–Ү

    жүҖжңүзҲ¬иҷ«з»§жүҝиҮӘ BaseCrawlerпјҢж”ҜжҢҒзј“еӯҳжңәеҲ¶гҖӮ

  3. ж•°жҚ®еә“ (database.py)

    SQLite ж•°жҚ®еә“пјҢеҢ…еҗ«пјҡ

    • professors иЎЁ: ж•ҷжҺҲеҹәжң¬дҝЎжҒҜ

    • papers иЎЁ: и®әж–Үе…ғж•°жҚ®

    жҸҗдҫӣе®Ңж•ҙзҡ„ CRUD ж“ҚдҪңе’Ңж•°жҚ®еҜјеҮәеҠҹиғҪгҖӮ

  4. ж•°жҚ®жЁЎеһӢ (models.py)

    Pydantic жЁЎеһӢе®ҡд№үпјҡ

    • Professor: ж•ҷжҺҲдҝЎжҒҜ

    • Paper: и®әж–ҮдҝЎжҒҜ

    • University: еӨ§еӯҰдҝЎжҒҜ

  5. е·Ҙе…·жЁЎеқ— (utils/)

    • logger.py: з»“жһ„еҢ–ж—Ҙеҝ—й…ҚзҪ®

    • helpers.py: йҖҡз”Ёиҫ…еҠ©еҮҪж•°

ж•°жҚ®жөҒзЁӢпғҒ

  1. зҲ¬еҸ–йҳ¶ж®ө

    User вҶ’ main.py crawl
   вҶ’ CSRankingsCrawler.fetch()
   вҶ’ Selenium жү“ејҖжөҸи§ҲеҷЁ
   вҶ’ и§Јжһҗ HTML жҸҗеҸ–ж•ҷжҺҲеҲ—иЎЁ
   вҶ’ Database.upsert_professor()
   вҶ’ SQLite дҝқеӯҳ

  2. и®әж–ҮиҺ·еҸ–йҳ¶ж®ө

    User вҶ’ main.py fetch-papers
   вҶ’ Database.list_professors()
   вҶ’ еҜ№жҜҸдҪҚж•ҷжҺҲ:
      ArxivCrawler.fetch(professor)
      вҶ’ arxiv.Search жҹҘиҜў
      вҶ’ и§Јжһҗиҝ”еӣһз»“жһң
      вҶ’ Database.upsert_paper()
   вҶ’ SQLite дҝқеӯҳ

  3. жҹҘиҜўйҳ¶ж®ө

    User вҶ’ main.py stats / list
   вҶ’ Database.get_stats() / list_professors()
   вҶ’ ж јејҸеҢ–иҫ“еҮә

й…ҚзҪ®пғҒ

еҪ“еүҚзүҲжң¬ж— йңҖй…ҚзҪ®ж–Үд»¶пјҢжүҖжңүеҸӮж•°йҖҡиҝҮе‘Ҫд»ӨиЎҢдј йҖ’гҖӮ

жңӘжқҘеҸҜиғҪзҡ„й…ҚзҪ®йЎ№пјҡ

  • ж•°жҚ®еә“и·Ҝеҫ„

  • зҲ¬иҷ«е»¶иҝҹж—¶й—ҙ
